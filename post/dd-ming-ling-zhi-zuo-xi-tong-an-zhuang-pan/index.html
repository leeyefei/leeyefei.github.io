dd命令制作系统安装盘

2020-03-23 1 min read # linux

linux下有很多工具可以制作启动盘，不过我们可以使用linux下的一条命令来完成-----dd

操作方法:

1 卸载你的U盘

假设你的u盘对应的设备是sdb1(可以使用
df查看或者fdisk)

sudo umount /
dev/sdb1

2 写入linux(ubuntu fedora .....)系统

sudo dd if=系统路径 of=/
dev/sdb

例如 sudo dd if=./ubuntu-13.04-beta2-desktop-amd64.iso of=/
dev/sdb
